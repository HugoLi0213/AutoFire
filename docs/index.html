AutoFireは、Windows向けの高度なキーボード自動化ツールです。
SendInput APIとPostMessage APIの2つの入力モードを搭載し、
DirectInput対応ゲームを含む様々なアプリケーションで動作します。

【 主要機能 】

⚙️

デュアル入力モード

SendInput API: ハードウェアレベルのスキャンコードを使用し、DirectInput対応ゲームで動作
PostMessage API: メッセージキューベースの入力方式

🌏

多言語UI対応

ワンクリックで言語切り替え:
• English (英語)
• 繁體中文 (台湾・香港)
• 简体中文 (中国大陸)

🎯

ウィンドウ選択

実行中のウィンドウを自動検出し、ドロップダウンメニューから選択可能。特定のアプリケーションのみをターゲットにできます。

⏱️

カスタマイズ可能

• トリガーキーの設定
• 出力キーの設定
• 間隔調整 (1ms〜10000ms)
• パススルーモード

💾

設定保存機能

すべての設定はJSON形式で自動保存され、次回起動時に復元されます。言語設定も保持されます。

🔒

安全設計

スレッドセーフな実装、エラーハンドリング、管理者権限での実行。オープンソースで透明性を確保。

Simple & Intuitive Interface

Key Features

🎮

Dual Output Modes

SendInput Mode: Hardware-level simulation with scan codes. Works with DirectInput games like AutoHotkey.

PostMessage Mode: Safer message queue simulation for window-specific targeting.

🪟

Smart Window Detection

Auto-detect all visible windows with dropdown selection. Real-time window list refresh with manual title input support.

🌐

Multi-Language UI

Switch between English, Traditional Chinese (繁體中文), and Simplified Chinese (简体中文) with one click.

⚙️

Flexible Configuration

Customize trigger key, output key, interval (1-1000ms), and pass-through mode. Settings auto-save.

🛡️

Safe & Legitimate

Uses standard Windows APIs (SendInput/PostMessage). No memory reading, no code injection, no DLL hooking.

💾

Portable & Lightweight

11 MB standalone executable. No Python installation required. Comprehensive test coverage included.

Frequently Asked Questions

Q: Does this work with all games?

A: Use SendInput mode for best compatibility. PostMessage mode doesn't work with DirectInput games.

Q: Will I get banned?

A: This uses legitimate Windows APIs like AutoHotkey. However, check your game's Terms of Service. Some games prohibit automation.

Q: Why do I need Administrator privileges?

A: Windows requires admin rights for global keyboard hooks to detect your trigger key.

Q: SendInput vs PostMessage - which should I use?

A: SendInput (default) works with more games. PostMessage is safer but less compatible.

Q: How do I switch languages?

A: Click the "EN/繁中" button in the top-right corner of the app.
